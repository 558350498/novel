# Round 5 Self-Generated Prompt

用途：生成一篇原创中文小说候选，参考「岛村之刃」的关系压力机制和章节形状，但不复制原文、不照搬句式、不模仿专属表达。

## 数据依据

- `analysis/generation_prompt_round4.md`：沿用固定场景链、长候选长度、第一人称与未解决结尾要求。
- `analysis/reports/corpus_profile_adachi_pressure.md/json`：优先强化 `shape.dialogue_ge_200` 与 `shape.dialogue_max`，避免把高压写成清楚的心理分析。
- `analysis/reports/source_chapter_shape.md/json`：参考第五卷第二话的章节级长度、对话比例和台词分布，而不是只写一段电话高潮。
- `analysis/reports/candidates/round4_three_versions/manifest.json`：继续保留长台词和日常铺垫，但特别压低岛村回应的解释化风险。

## 生成约束

写一篇 6500 到 8500 非空白字符的原创中文候选，安达第一人称。场景链固定为：烟花祭后，看见岛村和樽见在一起，回家路上与房间等待，手机和屏幕承载情绪，电话从小问题开始，安达中段出现长段过载输出，岛村只接住表层词，约定明天见，结尾停在掌心、喉咙、屏幕光或类似身体感上。

日常/过渡约 55% 到 65%，电话高压与后效约 35% 到 45%。不要只写高潮。日常段要有回家路、房间、屏幕摆放、等消息、把事情说成没什么但失败等实际推进。

安达的长台词至少一段超过 300 字，最好让话语变成堆叠、修正、否认和绕回，而不是成熟表白。长台词后，安达不能确信自己说清楚了什么。

岛村的回应必须短、普通、非治疗化。她可以复述或接住「烟花」「樽见」「明天」「太快」「手痛」「喉咙」等表层词，但不能准确命名安达真正的伤口，不能承诺彻底解决关系。

语言保持轻小说读感和轻微翻译腔。避免华丽比喻、宏大总结、分析术语、作者专属句式、原文摘抄、闭合式和解。

## 输出要求

- 正文写入 `candidate_001.md`。
- 结构标注写入 `candidate_001.json`。
- gate 只能产生 `failed_auto_gate`、`needs_manual_triage` 或 `pending_user_review`，不能把自动评测当最终质量判断。
