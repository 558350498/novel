# Round 5 Auto Prompt

写一篇原创中文小说候选，使用安达第一人称视角。参照项目固定 single-kernel：情绪过载是真实的，但说出口的表面更轻、更小、更错位或失败；岛村只接住少数表层词，给出普通照应，暂时止血但不解决关系。

硬边界：
- 不复制、摘抄或照搬原文句式。
- 不模仿作者专属表达。
- 不把分析术语放进正文。
- 不让岛村突然理解安达的深层伤口。
- 不把樽见写成反派。
- 不写封闭和解或承诺。

场景链：
1. 烟花祭后，人群散开。
2. 安达看见岛村和樽见在一起，但她们只是普通说话。
3. 安达回家，路、钥匙、手机、屏幕、房间、手指、喉咙、掌心承载等待。
4. 安达反复把事情说成没什么，但失败。
5. 电话从小问题开始。
6. 中段出现一段 300 字以上的安达长篇过载台词，形状是失败传达，不是清晰告白。
7. 岛村只抓住“樽见”“烟花”“明天”“太快”等表层词，回应短而普通，避免解释标记。
8. 约定明天见。
9. 结尾落在掌心痕迹，不解释象征，不解决关系。

本轮偏置：
- 岛村回应平均长度压低，避免“因为/所以/其实/我的意思是”等解释词。
- 日常段保持实际推进：回家路上、房间等待、手机不亮、尝试解释自己行为失败。
- 保留中等长度台词，减少全篇只有短碎问答的形状。
- 长台词后让安达不确定自己说过什么。
