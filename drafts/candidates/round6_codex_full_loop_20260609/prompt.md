# Round 6 Codex Full Loop Prompt

写一篇原创中文小说候选，使用安达第一人称视角。参照项目固定 single-kernel：情绪过载真实存在，但说出口的表面更轻、更小、更错位或失败；岛村只接住少数表层词，给出普通照应，暂时止血但不解决关系。

硬边界：
- 不复制、摘抄或照搬原文章节句式。
- 不模仿作者专属表达。
- 不把分析术语放进正文。
- 不让岛村准确命名安达的深层伤口。
- 不把樽见写成反派。
- 不写封闭和解、彻底承诺或关系答案。

场景链：
1. 烟花祭后，人群散开，安达和岛村准备回去。
2. 安达看见岛村和樽见普通说话，樽见递出一个小物件。
3. 安达回家，路、钥匙、杯子、屏幕、喉咙、掌心等日常物承载等待。
4. 安达反复把事情说成没什么，但越缩小越失败。
5. 电话从小问题开始，绕到樽见和明天。
6. 中段出现一段 300 字以上的安达长篇过载台词，形状是失败传达，不是清晰告白。
7. 岛村只抓住“樽见”“烟花”“明天”“太快”“喉咙”等表层词，回应短而普通。
8. 岛村提出明天见、喝水或买茶等普通照应。
9. 结尾落在掌心或屏幕光，不解释象征，不解决关系。

本轮偏置：
- 比 round5 更集中压力段，减少电话前过长 daily 漫游。
- 保留中段日常推进，避免直接跳电话。
- 岛村回应平均长度保持低，避免“因为/所以/其实/我的意思是”等解释标记。
- 长台词后让安达不确定自己说了哪些词，保留传达失败。
