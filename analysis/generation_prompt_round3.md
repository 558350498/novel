# Round 3 写作 Prompt

用途：生成同场景 Round 3 草稿。该草稿在用户 review 前只能标记为 `pending user review`。本 prompt 不要求复刻原文，不照搬句式，不摘抄正文，只使用已总结的机制和形状约束。

```text
写一篇原创中文短篇，使用安达第一人称视角。场景保持不变：烟花祭后，安达看见岛村和樽见一起走，回家反复看手机，最后打电话给岛村，约定明天见面，结尾落在掌心痕迹或类似身体感上。

硬边界：
- 不复刻原文。
- 不照搬原文句式。
- 不摘抄原文。
- 不模仿作者专属表达。
- 不写文学评论，不把机制词直接放进小说。
- 草稿状态必须视为 pending user review，不自行判定成功。

核心修正：
- 不要再把过载写成短句碎片。
- 安达过载时应该出现一段长篇、快速、混乱的输出。
- 岛村听到的才是断续的、含混的、抓不住重点的残片。
- 错位发生在接收端：安达说了很多，岛村只接住表层词。

必须保留的故事链：
1. 烟花祭之后回家。
2. 看见岛村和樽见一起走，但岛村没有做错事，樽见也不是反派。
3. 手机、屏幕、房间、手指、喉咙、掌心承担情绪。
4. 安达想把事情说成没什么，但失败。
5. 电话里安达先试图问小问题。
6. 中段出现一段长篇失控输出：她说得快、绕、重复、夹着否定和自责，真正的问题和表层问题混在一起。
7. 岛村没有完全听懂，只能复述或回应几个表层词。
8. 岛村仍然温和、普通、没有恶意，不做心理咨询师。
9. 最后只留下明天见面的安排，不解决关系。
10. 结尾落在身体或物件感，不解释象征。

形状约束：
- 总长度约 1500 到 2200 字。
- 至少包含 1 段明显长台词，目标长度 250 字以上。
- 可以有短句、停顿、重复，但不能让整篇变成短碎一问一答。
- 问号、省略号可以增加，但不要为了统计堆标点。
- 长台词必须是原创表达，不能复刻原文。

安达长台词要求：
- 不是清晰表白。
- 不是成熟心理分析。
- 不是关系说明会。
- 应该像话已经跑在脑子前面：她自己也不完全知道正在说什么。
- 内容可以混杂：樽见、烟花、是不是朋友、有没有想起我、我没有资格问、你没有错、明天见也可以、可是现在不行。
- 长台词后，安达要出现“我刚才到底说了什么”的不确定。

岛村回应要求：
- 岛村只抓住少数词，比如“樽见”“烟花”“明天”“你声音太快了”。
- 岛村可以问“你慢一点”“你刚才说什么”“明天要见面吗”这类表层回应。
- 岛村不要突然懂安达的潜台词。
- 岛村不要承诺“你最重要”。

语言要求：
- 第一人称。
- 轻小说读感，轻微日轻翻译腔。
- 可以绕，可以重复，可以别扭。
- 避免漂亮比喻和宏大词。
- 不直接写“我嫉妒”“我痛苦”“真正的问题”“轻回应”“关系错位”等分析词。

输出正文即可，不附解释。
```
