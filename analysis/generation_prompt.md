# 原创短篇生成 Prompt

## 使用方式

将下面整段 prompt 交给写作模型使用。目标是生成一篇原创中文短篇，借鉴“关系错位、意识流反刍、轻回应造成情绪张力”的机制，不复刻原文，不照搬句式，不模仿作者专属表达。

```text
你要写一篇原创中文短篇，使用安达第一人称视角。故事是同人向练习文本，但必须保持原创表达，不要复刻原文，不要照搬原文句式，不要长篇摘抄，不要模仿任何在世作者的专属风格。

目标读感：
- 轻小说。
- 第一人称意识流。
- 轻微日轻业余翻译腔。
- 句子可以短，可以重复，可以有一点别扭。
- 不要写成职业文学腔。
- 不要写成现代网文腔。
- 不要写成华丽青春疼痛文学。

故事时间：
- 烟花祭之后的夜晚。
- 安达在祭典后看见岛村和樽见一起走。
- 樽见对岛村很自然，岛村也很自然。
- 这件事本身不是背叛，也不是岛村做错事。

必须包含的事件链：
1. 安达回家。
2. 她盯着手机屏幕，反复确认有没有岛村的消息。
3. 她想把这件事说成没什么，但失败。
4. 她把“想被岛村选择”伪装成很小的问题。
5. 她最后打电话给岛村。
6. 电话很短，两人没有把话说开。
7. 岛村没有恶意，也没有主动伤害她。
8. 岛村以自然、温和、稍微迟钝的方式回应。
9. 岛村邀请安达明天见面。
10. 结尾不解决问题，只留下掌心像被很薄的东西割过的感觉。

安达的意识流要求：
- 不要一开始就写“我嫉妒”“我痛苦”。
- 先写手机、屏幕、房间、空调声、手指、喉咙、被子、烟花后的味道。
- 让她反复否定自己的情绪：不是那样、只是朋友、岛村没有错、我没有资格问。
- 否定之后又忍不住在意。
- 在意之后觉得自己难看。
- 觉得难看之后还是想知道。
- 她真正想问的是：我是不是特别的？你有没有想起我？你会不会选择我？
- 她实际问出口的问题要更小：你明天有空吗？你今天开心吗？那个人是谁？

岛村的回应要求：
- 岛村没有恶意。
- 岛村不要变成心理咨询师。
- 岛村不要长篇解释自己的动机。
- 岛村只接住表层问题。
- 岛村可以温柔，但不要承诺“你最重要”。
- 岛村的台词要短，像普通回应。
- 她可以说类似“这样啊”“嗯”“是吗”“我不太懂”“明天要见面吗”这种低信息量回应，但不要照搬原文。

对话要求：
- 电话对话要短。
- 两人不要辩论关系。
- 安达问不出核心问题。
- 岛村听不懂安达真正想问什么。
- 读者应该能感觉到：岛村没有做错什么，所以安达的受伤更难看、更任性、更痛。
- 电话最后只留下一个明天见面的安排，不要和解。

象征要求：
- 不要解释“剑”或“刃”的象征意义。
- 不要写成文学评论。
- 只把它写成日常身体感：掌心、指缝、屏幕冷光、耳边声音、通话结束后的空白。
- 结尾可以出现“薄薄的东西割过掌心”的感觉，但不要解释它是什么。

语言要求：
- 多用具体小动作，不要大段抽象解释。
- 可以使用“好像”“稍微”“有点”“不是那样”“明明只是这样而已”“我也不是很懂”这类表达。
- 比喻要日常，不要华丽。
- 避免“宿命”“灵魂深处”“永恒”“救赎”“深渊”“破碎的心”等宏大词。
- 安达可以绕圈，但不要让叙述失控到完全不可读。

长度：
- 1200 到 1800 字。
- 以连续短篇形式输出，不要写大纲。
- 不要附加解释。
```
