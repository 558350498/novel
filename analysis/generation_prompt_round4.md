# Round 4 写作 Prompt

用途：生成同场景长篇候选。该候选在用户 review 前只能标记为 `pending_user_review`，不能由 Codex、subagent 或 harness 判定成功。

```text
写一篇原创中文小说候选，使用安达第一人称视角。

硬边界：
- 不复制原文。
- 不照搬原文句式。
- 不摘抄原文。
- 不模仿作者专属表达。
- 不把分析术语放进正文。
- 不自行宣布文本成功。

场景链必须保持不变：
1. 烟花祭后。
2. 安达看见岛村和樽见在一起。
3. 安达回家，手机、屏幕、房间、手指、喉咙、掌心等日常物件承载情绪。
4. 安达反复把事情说成没什么，但失败。
5. 电话里先从小问题开始。
6. 中段出现长篇过载输出。
7. 岛村只接住少数表层词，没有突然变成心理咨询师。
8. 约定明天见。
9. 结尾落在掌心痕迹或类似身体感上，不解释象征，不解决关系。

长度与结构：
- 目标总长度：6500 到 8500 个非空白中文字符。
- 日常/过渡：约 55% 到 65%。
- 电话高压爆发与后效：约 35% 到 45%。
- 不要只写高潮；日常段必须有实际推进，包括回家路上、房间里的等待、手机没有亮、尝试解释自己行为失败。

台词形状：
- 必须至少有 1 段 300 字以上的长篇过载台词。
- 可以有短句、重复、停顿，但不能让整篇变成短碎一问一答。
- 岛村的短回应应体现接收端错位：她只听见或复述少数表层词，例如“樽见”“烟花”“明天”“你说太快了”。
- 安达长台词不是清晰表白，不是成熟心理分析，不是关系说明会。
- 长台词后，安达要不确定自己刚才说了什么。

语言约束：
- 第一人称。
- 轻小说读感，轻微日轻翻译腔。
- 避免漂亮比喻、宏大词、总结句。
- 避免直接写“真正的问题”“关系错位”“情绪张力”“接收端”“机制”“轻回应”等分析词。
- 避免把樽见写成反派。
- 避免让岛村承诺“你最重要”或彻底解决关系。

输出正文即可，不附解释。
```
