# 可迁移写作规则

## 情绪张力规则

1. 不要从“安达很嫉妒”开始。先写她遇到一个具体细节：未读消息、昵称、岛村和别人并肩、来电记录为空、手指停在拨号键上。
2. 不要让事件本身很大。事件越普通，安达的受伤越难以辩护，文本越有张力。
3. 不要让岛村做错事。岛村只是自然地和别人来往、自然地回答、自然地没有意识到安达在意什么。
4. 安达的核心痛点不是“别人出现”，而是“岛村有我不知道的部分，而且那个部分看起来也很自然”。
5. 关系冲突不要写成背叛审判，要写成特殊性需求无法被承认。

## 安达意识流规则

1. 每个情绪段落从物件或动作进入：手机、屏幕、空调、枕头、手指、喉咙、掌心、房间灯光。
2. 内心循环使用“否定 → 在意 → 自责 → 再在意”的路径。
3. 让她知道自己无理，但不要因此停止受伤。
4. 让她把重问题伪装成轻问题：不要直接问“你会选我吗”，而是问“明天有空吗”“你开心吗”“那个人是谁”。
5. 写她的句子时允许重复和撤回：她不是在发表感想，而是在压住即将失控的东西。
6. 崩溃前句子要窄，崩溃时句子可以突然变长、问题成串、名字重复。
7. 不要把她写成稳定分析自己依恋模式的人。她只能在事后意识到一点点，而且意识到也未必能改变。

## 岛村回应规则

1. 岛村的回答要轻，不要冷酷。她可以关心，但不要立刻理解情绪规模。
2. 岛村不要解释自己为什么和别人同行。她只需要普通地承认事实。
3. 岛村不要长篇安抚。她的修补方式应是短句、实际安排、轻微道歉或自然换页。
4. 岛村可以温柔，但温柔停在当下照顾，不要变成情绪责任承诺。
5. 岛村说出的刺痛话要短。越像随口判断，越像刃。
6. 岛村不要主动定义关系。她一旦把关系说得太清楚，安达的悬浮感就消失了。

## 对话规则

1. 电话对话开头要短，双方都只接住表层。
2. 不要让两人辩论关系定义。
3. 安达问不出核心问题时，使用重复事实替代推进。
4. 岛村听不懂潜台词时，不要让她突然变聪明。
5. 读者应该比岛村更懂安达，也比安达更懂岛村没有恶意。
6. 对话结束不能解决问题。最多给一个明天见面的安排，作为临时止血。

## 象征规则

1. 不要写“这把剑象征距离”。把“刃”分散进薄痛、冷光、耳边声音、手指、屏幕、掌心。
2. 刃不是岛村主动挥出的，而是安达靠近岛村的轻关系边界时被割到。
3. 象征只出现为感觉，不出现为解释。
4. 结尾如果需要留下“刃”，写成身体残留感，而不是主题总结。

## 日常物件规则

1. 优先写低调物件：手机屏幕、房间灯、被子边缘、空调声、手心汗、通话键。
2. 物件不要装饰场景，要承担没说出口的情绪。
3. 身体感要局部：喉咙、胃、眼底、指尖、掌心、胸口。
4. 比喻要日常、略笨拙，不要华丽。
5. 普通东西突然变刺，才是目标效果。

## 中文读感规则

1. 句子可以短，也可以重复。
2. 允许轻微翻译腔：这样啊、是吗、稍微、有点、不太懂、明明只是这样而已。
3. 不要写成职业文学腔。
4. 不要写成现代网文腔。
5. 不要让每个句子都很顺滑。安达的意识应该有卡顿、绕路和临时改口。

## 失败模式

1. 安达一开场就说清楚“我嫉妒你和别人一起去祭典”。
2. 岛村解释自己只是普通朋友，并温柔保证安达最重要。
3. 电话里两人把关系说开。
4. 结尾出现彻底安心或明确和解。
5. “刃”被写成显眼文学符号。
6. 比喻密度过高，日常物件被挤掉。
7. 台词像心理分析，而不是普通高中生勉强沟通。
