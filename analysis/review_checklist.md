# 成稿审稿清单

## 版权与风格边界

- [ ] 成稿没有复刻原文句式。
- [ ] 成稿没有照搬原文段落结构。
- [ ] 成稿没有长篇摘抄原文。
- [ ] 成稿没有要求或表现为模仿作者专属表达。
- [ ] 成稿是原创表达，只迁移关系机制和叙事方法。

## 安达视角

- [ ] 安达没有一开头就清楚说出“我嫉妒”“我痛苦”。
- [ ] 情绪先落在手机、屏幕、房间、手指、喉咙、掌心等具体对象上。
- [ ] 文中存在“否定 → 在意 → 自责 → 再在意”的循环。
- [ ] 安达知道岛村没有错，但仍然受伤。
- [ ] 安达真正想问的问题和实际问出口的问题之间存在差距。
- [ ] 安达的内心独白有绕圈、撤回、重复，而不是稳定理性分析。

## 岛村回应

- [ ] 岛村没有主动恶意伤害安达。
- [ ] 岛村没有变成心理咨询师。
- [ ] 岛村没有长篇解释自己的动机。
- [ ] 岛村主要接住表层问题，而不是直接翻译安达的潜台词。
- [ ] 岛村的温柔停在当下照顾或实际安排，没有承诺“你最重要”。
- [ ] 岛村的轻回应能造成错位，而不是显得冷酷。

## 对话结构

- [ ] 电话对话足够短。
- [ ] 对话不是辩论，也不是关系说明会。
- [ ] 安达没有顺利问出核心问题。
- [ ] 岛村没有完全听懂安达真正想问什么。
- [ ] 读者能读出潜台词，但角色没有完全读出。
- [ ] 明天见面的邀请只是临时止血，不是解决问题。

## 情绪张力

- [ ] 情绪张力来自小动作和关系错位，而不是大段解释。
- [ ] 岛村没有做错事这一点被保留。
- [ ] 安达的受伤因此显得任性、难看，但也真实。
- [ ] 文本没有把樽见写成反派。
- [ ] 文本没有把岛村写成背叛者。
- [ ] 文本没有让安达获得完全正当化。

## 象征与物件

- [ ] “刃”没有被解释成固定象征。
- [ ] 结尾的痛感落在身体或物件上，而不是主题宣言。
- [ ] 手机、屏幕、通话、掌心、房间等日常物件承担了情绪。
- [ ] 比喻保持日常尺度。
- [ ] 没有出现过多宏大、华丽、抽象的比喻。

## 中文读感

- [ ] 文本有轻小说读感。
- [ ] 文本有轻微日轻业余翻译腔。
- [ ] 句子可以短，可以重复，但仍能读懂。
- [ ] 没有写成成熟散文。
- [ ] 没有写成现代网文爽感。
- [ ] 没有写成国产青春疼痛文学。

## 结尾检查

- [ ] 结尾没有和解。
- [ ] 结尾没有彻底说开。
- [ ] 结尾没有让安达安心。
- [ ] 结尾没有让岛村给出明确承诺。
- [ ] 结尾只留下一个未解决的身体感或物件感。
- [ ] 如果结尾解决了问题，就是失败。

## 自动评估建议

运行：

```powershell
python tools/style_evaluator.py --mode draft drafts/current.md --output analysis/reports/current.md
```

先按 `analysis/lexicon_taxonomy.md` 的粒度理解信号：taxonomy 是泛化语言学维度，下面这些是当前项目 gate 标签。

重点看这些 gate 风险：

- `情绪表达直白度` 是否过高。
- `含混与自我修正` 是否过低。
- `日常物件承载` 是否过低。
- `对话占比` 是否过高。
- `岛村回应解释化` 是否偏高。
- `结尾封闭化` 是否命中。
- `过度文学化风险` 是否命中。
