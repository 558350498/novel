# Delta v1 语料切片

这里保存 Delta 相对距离评估器使用的参照语料配置。

## 文件

- `slices.json`：参照组配置，只记录文件路径和行号范围，不复制正文。

## 行号规则

- `start_line`：包含该行。
- `end_before_line`：不包含该行。
- 没有行号字段时，读取整份文件。

## 当前参照组

- `adachi_pressure`：第五卷「岛村之刃」关系失衡核心段。
- `adachi_daily`：第五卷「来自蔚蓝」关系错位前置段。
- `shimamura_view`：第二卷「岛村 前往健身房」岛村平稳叙述段。
- `analysis_docs`：四份核心分析文档，用于检测生成稿是否靠近分析腔。

## 使用边界

Delta 只比较字符 unigram、字符 bigram、标点频率等表层特征的相对距离。它不判断文本质量，不输出作者相似度，也不作为复刻目标。
