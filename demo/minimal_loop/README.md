# Minimal Loop Demo

This directory documents the smallest expected process shape:

```text
candidate_001.md + candidate_001.json
-> candidate_001_gate.json
-> rewrite_plan.json
-> candidate_002.md + candidate_002.json
-> user_review_ledger_entry.json
```

The files are synthetic and intentionally tiny. They are not full-candidate evidence and do not prove literary quality.

Use this demo to explain artifact handoff rules, not to tune gate thresholds.
